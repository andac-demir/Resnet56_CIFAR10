# ResNet18 CIFAR-10

### Prerequisites
```
-numpy (1.17.3)
-fire (0.2.1)
-torch (1.3.1)
-torchvision (0.4.2)
-matplotlib (3.1.1)
```

### Downloading CIFAR Dataset
```
python data_processing.py
```

### Training Model
```
python NN_classifier.py train
```

### Testing Model
```
python NN_classifier.py test <image_path>
```
